**Version:** 1.0 (Architecture Freeze)

---

# 1. Introduction

The BLIE Memory Architecture defines how the Buddy Learning Intelligent Engine remembers, understands, and adapts to each student's learning journey.

The purpose of memory is not simply to store conversation history.

Memory enables BLIE to understand:

* What the student has learned.
* What the student understands.
* What the student struggles with.
* How the student learns best.
* What knowledge should be reinforced next.

---

# 2. Memory Vision

To create a lifelong academic memory that grows with the student from the first semester until graduation.

The student's learning history becomes an intelligent foundation for continuous personalized guidance.

---

# 3. Memory Architecture Principle

BLIE memory consists of four layers:

```text
Short-Term Memory

        ↓

Learning Session Memory

        ↓

Academic Memory

        ↓

Personal Learning Intelligence
```

Each layer has a different purpose.

---

# 4. Short-Term Memory

## Purpose

Supports the current learning interaction.

Example:

Student:

"Explain this formula."

BLIE remembers:

* Current conversation.
* Current topic.
* Current uploaded material.
* Current question context.

---

Characteristics:

* Temporary.
* Fast access.
* Automatically expires.

---

# 5. Learning Session Memory

## Purpose

Captures meaningful information during a study session.

Examples:

Student studies:

"Database Normalization"

BLIE records:

* Topics reviewed.
* Questions asked.
* Concepts explored.
* Confusion points.
* Learning outcomes.

---

Example:

```text
Study Session

Subject:
Database Systems

Topic:
Normalization

Student Difficulty:
Functional Dependency

Recommended Review:
1NF → 2NF → 3NF
```

---

# 6. Academic Memory

## Purpose

Maintains long-term educational history.

This includes:

* Completed subjects.
* Learned concepts.
* Academic progress.
* Assessment history.
* Knowledge growth.

Academic memory connects directly with PLKG.

---

Example:

```text
Semester 1

Programming Fundamentals

Concepts:
- Variables
- Functions
- Loops


Semester 2

Data Structures

Requires:
- Functions
- Algorithms
```

---

# 7. Personal Learning Intelligence Memory

This is the highest-level memory layer.

BLIE learns about the student's learning characteristics.

Examples:

## Learning Strengths

Student understands:

* Visual explanations.
* Practical examples.
* Step-by-step reasoning.

---

## Learning Challenges

Student struggles with:

* Abstract concepts.
* Mathematical derivations.
* Memorization-heavy topics.

---

## Learning Preferences

Student prefers:

* Short explanations.
* Detailed explanations.
* Examples first.
* Theory first.

---

# 12. Memory Update Pipeline

Every meaningful interaction follows:

```text
Student Interaction

↓

↓
```

# 13. Memory Importance Classification

Not every interaction becomes permanent memory.

BLIE classifies:

## Temporary

Example:

"Explain this sentence."

No long-term storage.

---

## Useful

Example:

"Student struggles with recursion."

Store learning insight.

---

## Important

Example:

"Student repeatedly struggles with programming logic."

Create long-term learning pattern.

---

# 14. Memory Privacy Architecture

Student memory is private.

Protected data:

* Learning weaknesses.
* Study behaviour.
* Academic history.
* AI conversations.

BLIE can use memory only for that student's learning assistance.

---

# 15. Memory Optimization

Long-term memory must remain efficient.

The system uses:

* Summarization.
* Importance scoring.
* Knowledge consolidation.
* Duplicate removal.

The goal:

More intelligence with less storage.

# Architecture Freeze

BLIE Memory Architecture defines AI Study Buddy as a true learning companion.

The memory system:

1. Preserves meaningful learning history.
2. Understands student progress.
3. Personalizes educational support.
4. Enriches the Personal Learning Knowledge Graph.
5. Evolves throughout the student's university journey.

**BLIE should know the student's learning journey as a trusted study companion, while keeping the student in full control of their academic memory.**
